# Visual Novel Final

Браузерная визуальная новелла на React + Electron.

## Как запустить локально

1. Установите зависимости:
   ```
   npm install
   ```
2. Запустите для разработки:
   ```
   npm run start:web
   ```
   Откройте [http://localhost:3000](http://localhost:3000)

3. Для production-сборки:
   ```
   npm run build:web
   ```
